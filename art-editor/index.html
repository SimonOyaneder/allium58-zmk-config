Exportar a C

Genera el archivo completo en formato shield: rotado 90° horario y con los bits invertidos (tinta = bit 0, paleta index 0 = blanco), tal como lo espera el pipeline de zmk main + LVGL 9. Pega el resultado tal cual en boards/shields/nice_view_gem/assets/.

Importar C existente

Pega el contenido completo de assets/moon_nika.c o night_mountains.c: detecta el formato shield (rotado + bits invertidos) por el descriptor y lo endereza para editar. Un array plano vertical también funciona (usa el ancho/alto del panel «Lienzo»).
